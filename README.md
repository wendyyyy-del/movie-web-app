# 🎬 FourFrame - Movie Web App

FourFrame is a full-stack movie review web application built with **React (frontend)** and **Flask (backend)**.

## 🔧 Technologies Used

### Frontend
- React
- Axios
- React Router DOM

### Backend
- Flask
- Flask SQLAlchemy
- Flask-JWT-Extended
- SQLite
- Flask-Migrate
